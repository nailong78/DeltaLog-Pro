**DeltaLog-Pro** 是一款专为《三角洲行动》玩家打造的云端战术记录与收益分析平台。它融合了 **iOS 26 液态玻璃 (Liquid Glass)** 极简美学，通过实时数据看板与 AI 战术引擎，帮助干员精准掌控每一场行动的盈亏与风险。

---

## ✨ 核心特性

* **液态玻璃美学**：采用超高采样率的高斯模糊、液态流动边框及动态响应布局，复刻 iOS 26 的视觉质感。
* **全量数据看板**：
    * **环形撤离率**：直观展示当前地图的生存现状。
    * **收益波动趋势**：基于 Chart.js 的贝塞尔曲线，可视化近期物资价值走向。
* **AI 战术分析引擎**：基于撤离率、收益均值及备注关键词（如“老六”、“出红”），自动生成针对性的动态战术指导。
* **云端实时存档**：基于 **Supabase** 无服务器架构，确保行动记录多端同步，永不丢失。
* **多维度筛选**：支持按地图（零号大坝、长弓溪谷、航天基地等）快速切片分析，锁定最高收益区域。

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone [https://github.com/nailong78/DeltaLog-Pro.git](https://github.com/nailong78/DeltaLog-Pro.git)
### 2. 数据库准备 (Supabase)
为了使云端存储生效，请在 Supabase SQL Editor 中运行以下命令创建表：
create table delta_logs (
  id uuid default gen_random_uuid() primary key,
  created_at timestamptz default now(),
  mission_name text,
  operator text,
  result text,
  loot_value int8,
  notes text
);
3. 配置 API
在 index.html 的 <script> 部分替换你的 SUPABASE_URL 和 SUPABASE_KEY 即可完成对接。
