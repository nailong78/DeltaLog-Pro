# DeltaLog-Pro 📊

> **基于 iOS 26 液态玻璃美学的《三角洲行动》云端战术记录与收益分析平台**

**DeltaLog-Pro** 是一款专为《三角洲行动》玩家设计的沉浸式行动记录工具。它不仅能帮助你精准统计每一场行动的物资收益，更内置了基于统计学逻辑的 **AI 战术指导引擎**，协助你在复杂多变的战场中制定最佳决策。

---

## ✨ 核心特性

### 1. iOS 26 液态玻璃系统设计 (Liquid Glass UI)

* **高斯模糊 (Gaussian Blur):** 采用 `backdrop-filter: blur(40px)` 实现深度的景深散焦效果。
* **物理流动感:** 筛选器带有流光动效 (Liquid Sheen)，模拟液体表面张力产生的高光。
* **自适应布局:** 完美适配灵动岛与刘海屏的安全区域排版，提供原生级 App 体验。

### 2. 实时云端同步

* **Supabase 后端驱动:** 实现多端数据毫秒级同步，确保你的行动记录永不丢失。
* **无感存储:** 采用异步提交技术，确保上传存档时页面依然保持流畅。

### 3. AI 战术分析引擎

* **风险评估:** 根据近期撤离成功率，自动调节战术建议（从“潜行绕路”到“高风险博弈”）。
* **语义分析挖掘:** 自动识别备注中的“老六”、“出红”等关键词，预测当前的物资爆发窗口期或危险点位。

### 4. 数据可视化分析

* **环形进度条:** 直观展示当前区域的生存率。
* **贝塞尔曲线趋势图:** 使用 Chart.js 绘制近 10 场收益波动，捕捉财富增长趋势。

---

## 🛠️ 技术实现

### 前端技术栈

* **样式引擎:** CSS 变量控制下的高级液态玻璃特效。
* **图表库:** [Chart.js](https://www.chartjs.org/) (响应式配置)。
* **交互:** 纯原生 JavaScript 实现的高效 DOM 渲染。

### 后端服务

* **数据库:** Supabase (PostgreSQL)。
* **API:** Supabase-js SDK 实时连接。

## 🚀 快速开始

1. **克隆项目**
```bash
git clone https://github.com/nailong78/DeltaLog-Pro.git

```


2. **配置数据库**
在 `index.html` 中替换你的 Supabase 配置信息：
```javascript
const SUPABASE_URL = '你的URL'; 
const SUPABASE_KEY = '你的KEY';

```


3. **运行**
由于是静态 HTML 项目，直接双击 `index.html` 或通过 Live Server 预览即可。

## 📋 数据表结构参考

若要自行搭建后台，建议的 `delta_logs` 表结构如下：

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| `id` | uuid | 唯一标识 (主键) |
| `created_at` | timestamp | 记录时间 |
| `operator` | text | 使用的干员名称 |
| `mission_name` | text | 地图与难度级别 |
| `result` | text | 撤离成功/失败 |
| `loot_value` | int4 | 收益价值 (数字) |
| `notes` | text | 备注信息 |
